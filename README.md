# EnergyAir

Tiny ruby bot to win [energy air](https://energy.ch/air) tickets.

## Installation

Install it:
(Make sure that you have [phantomjs](http://phantomjs.org/) installed)

    $ gem install energy_air

## Usage

	$ energy_air <telephone_number>
